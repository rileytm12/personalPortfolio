2D Kalman filter — rocket position & velocity estimation

Model-based prediction and sensor fusion to estimate a rocket's horizontal and vertical motion from noisy flight data in a simplified guidance, navigation, and control (GNC) setting.

  • MATLAB
  • Kalman filtering
  • State estimation
  • GNC
GitHub repository

Overview

This project implements a 2D Kalman filter to estimate a rocket's position and velocity in both horizontal and vertical directions during flight using noisy sensor data and a simplified physics-based model. The system fuses acceleration inputs with position measurements to produce a more accurate state estimate than raw sensor readings alone.

The primary objective was to develop a practical understanding of Kalman filtering by combining implementation with theory—building the filter in code while deriving and documenting the underlying mathematics. This demonstrates how model-based prediction and sensor fusion reduce noise and improve state estimation in guidance, navigation, and control (GNC) systems.

My work on the project

Kalman filter design & implementation:

Designed and implemented a discrete-time 2D Kalman filter to estimate position and velocity in both x and y directions using acceleration as the control input and position as the measurement.

System modeling:

Developed a kinematic model of rocket motion in two dimensions and constructed the corresponding state-space representation (F, B, H matrices).

Noise modeling & tuning:

Simulated realistic sensor noise for acceleration, velocity, and position measurements. Tuned process noise (Q) and measurement noise (R) to balance model predictions with sensor reliability.

Simulation & validation:

Generated noisy flight data and validated filter performance by comparing estimated states against ground truth, sensor measurements, and model predictions.

Software & architecture

The system follows a standard discrete-time estimation pipeline: 

[ acceleration input ] → [ prediction (F, B) ] → [ covariance update (P, Q) ]
                                              ↓
[ position measurement ] → [ correction (K, H, R) ] → [ updated state ]
  • Prediction step propagates position and velocity using system dynamics
  • Update step corrects estimates using position measurements
  • Covariance propagation tracks uncertainty over time

Estimation approach

State definition:

The system estimates a 4D state vector:

[ x, vx, y, vy ]T

Prediction model:

Uses constant-velocity kinematics with acceleration input:

xk+1 = F xk + B ak

Covariance propagation:

Uncertainty increases during prediction due to model simplifications (e.g., no drag or thrust variation).

Measurement update:

Position measurements in both x and y are used to correct the predicted state.

Kalman gain intuition:

  • High gain → trust measurements more
  • Low gain → trust model prediction more

Integration & testing

  • Simulated 2D rocket flight with Gaussian noise applied to acceleration, velocity, and position
  • Evaluated filter performance across both axes independently and jointly
  • Analyzed error growth through integration (acceleration → velocity → position)
  • Studied sensitivity to Q and R tuning

Results & lessons learned

  • The Kalman filter significantly reduced error in both position and velocity compared to raw sensor data and the physics-based model.
  • Error growth from integration was effectively mitigated through sensor fusion.
  • Proper tuning of process and measurement noise is critical for stability and accuracy.
  • Demonstrated how even a simplified 2D model can yield strong estimation performance when combined with filtering.

This project reinforces the importance of state estimation techniques in real-world GNC systems, where multi-dimensional motion and noisy sensors are unavoidable.

1D Kalman filter — rocket altitude & velocity

Model-based estimation and sensor fusion for noisy flight data in a simplified guidance, navigation, and control (GNC) setting.

Overview

This project implements a 1D Kalman filter to estimate a rocket’s altitude and velocity during flight using noisy sensor data and a simplified physics model. The system fuses acceleration inputs with altitude measurements to produce a more accurate state estimate than raw sensor readings alone.

The primary objective was to develop a strong, practical understanding of Kalman filtering by combining both implementation and theory—building the filter in code while formally deriving and documenting the underlying mathematics. This dual approach highlights how model-based prediction and sensor fusion work together to reduce noise and improve state estimation in guidance, navigation, and control (GNC) applications.

My work on the project

Kalman filter design & implementation:

Designed and implemented a discrete-time Kalman filter to estimate altitude and velocity using acceleration as an input and altitude as a measurement.

System modeling:

Developed a simplified kinematic model of rocket motion and constructed the corresponding state-space representation (F, B matrices).

Noise modeling & tuning:

Modeled realistic sensor noise for both accelerometer and altimeter data and tuned process noise (Q) and measurement noise (R) to balance model predictions with sensor trust.

Simulation & validation:

Simulated noisy flight data and validated filter performance by comparing estimated states against ground truth and raw measurements.

Software & architecture

The system follows a standard discrete-time state estimation pipeline: 

[ acceleration input ] → [ state prediction (F, B) ] → [ covariance update (P, Q) ]
                                             ↓
[ altitude measurement ] → [ correction (K, H, R) ] → [ updated state estimate ]
  • Prediction step uses system dynamics and control input
  • Update step incorporates sensor measurements
  • Covariance propagation tracks uncertainty over time

Estimation approach

State definition:

The system estimates a 2D state vector consisting of altitude and velocity.

Prediction model:

Uses constant-velocity kinematics with acceleration input:

xk+1 = F xk + B ak

Covariance propagation:

Uncertainty increases during prediction due to model simplifications (e.g., neglecting drag and thrust variation).

Measurement update:

Altitude measurements are used to correct the predicted state, with the Kalman gain dynamically adjusting trust between model and sensor.

Kalman gain intuition:

  • High gain → trust measurements more
  • Low gain → trust model prediction more

Integration & testing

  • Simulated rocket flight with injected Gaussian noise in both acceleration and altitude measurements
  • Evaluated filter stability and convergence over time
  • Observed behavior under different noise assumptions (Q vs R tuning)

Results & lessons learned

  • Kalman filtering significantly reduced noise in both altitude and velocity estimates compared to raw sensor data.
  • Proper tuning of process and measurement noise is critical to achieving stable and accurate estimates.
  • Demonstrated how even a simplified model can produce strong results when combined with sensor fusion.
  • Reinforced the importance of estimation techniques in real-world GNC systems where sensors are inherently noisy.

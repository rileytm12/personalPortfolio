Overview

This project is a personal portfolio website built using HTML, Tailwind CSS, and JavaScript, showcasing aerospace engineering work and technical experience. Development was made using AI-assisted workflows through Cursor with Claude Opus 4.7.

My work on the project

  • Designed and implemented a responsive multi-section portfolio UI
  • Used Tailwind CSS for modern styling and layout
  • Built interactive navigation with JavaScript
  • Integrated AI (Cursor + Claude) to streamline coding, debugging, and design iteration

Key features

  • AI-assisted development workflow: Faster iteration and refinement using Cursor + Claude Opus 4.7
  • Responsive design: Clean UI/UX across desktop and mobile
  • Interactive navigation: Sticky header and mobile menu
  • Structured sections: Projects, experience, and contact

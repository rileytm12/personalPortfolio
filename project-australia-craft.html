Overview

Australia Craft is a C++ survival game that simulates exploration, resource management, and decision-making in a dynamically generated environment. Players navigate different biomes, gather resources from animals and plants, and manage survival mechanics such as risk and scarcity.

The project emphasizes object-oriented design and system integration, combining file-driven world generation with interactive gameplay logic.

My work on the project

  • Designed and implemented the core game engine in C++, including the main game loop and player interaction systems
  • Built a data-driven architecture that reads items, animals, and plants from external .txt files
  • Implemented class-based systems for Items, Animals, Plants, and Game management
  • Developed biome-based spawning logic using randomness and rarity weighting
  • Created an interactive combat system (`fightAnimal`) using probability and user input
  • Implemented resource gathering mechanics (`foragePlants`) with stochastic outcomes
  • Integrated inventory and item lookup systems for dynamic gameplay progression

Key features

  • File I/O world generation: Game content dynamically loaded from text files
  • Randomized encounters: Animals and plants selected based on biome and rarity
  • Probabilistic combat system: Risk-based gameplay tied to player decisions
  • Modular OOP design: Clean separation between game systems and data models
  • Replayability: Random elements ensure unique runs each playthrough

Technologies used

  • C++
  • Standard Template Library (STL)
  • File I/O (`fstream`)
  • Object-Oriented Programming (OOP)

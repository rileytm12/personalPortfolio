Overview

This project is a collection of MATLAB-based state-space modeling and simulation studies focused on dynamic systems in aerospace applications. The work combines physics-based modeling, numerical integration, and optimization to analyze system behavior under varying conditions.

Projects include a nonlinear water-bottle rocket simulation and a multi-body orbital mechanics problem, demonstrating the application of state-space methods to both atmospheric and spaceflight systems.

My work on the project

  • Developed multiple state-space models representing nonlinear dynamic systems
  • Implemented ODE45-based numerical integration for time-evolving simulations
  • Built a 7-state rocket model including position, velocity, mass, and internal air dynamics
  • Designed a multi-body gravitational model for Earth–Moon–spacecraft interactions
  • Implemented event detection to terminate simulations based on physical conditions (collision with Earth/Moon)
  • Applied optimization (fminsearch) to minimize spacecraft return time
  • Created parameter sweep analyses (pressure, drag, volume, launch angle) to study system sensitivity
  • Validated simulations against provided datasets and analyzed model accuracy

Key features

  • Nonlinear dynamics modeling: State-space formulations of physically realistic systems
  • Multi-phase simulation: Water thrust, air thrust, and ballistic flight phases
  • Parameter sensitivity studies: Visualization of performance trends across input ranges
  • Optimization loop: Trajectory tuning for orbital mechanics problems
  • Event-driven stopping conditions: Physical realism via simulation termination logic
  • Data validation: Comparison against known solutions for verification

Technologies used

  • MATLAB
  • ODE45 (numerical integration)
  • Nonlinear state-space modeling
  • Optimization (fminsearch)
  • Data visualization and analysis

Overview

This project focuses on the integration and validation of an Attitude Determination and Control System (ADCS) for a multi-university CubeSat mission (SWARM-EX), involving CU Boulder, Georgia Tech, and Stanford. The work centers on implementing a commercial control system (XACT-15) and verifying its ability to meet mission pointing and stability requirements in Low Earth Orbit.

The project combines spacecraft dynamics, hardware-in-the-loop (HIL) simulation, and system-level integration to evaluate ADCS performance under realistic orbital conditions. Testing leverages real flight hardware, high-fidelity simulation environments, and telemetry analysis to ensure reliable operation within a fully integrated spacecraft system.

My work on the project

  • Integrated the XACT-15 ADCS module into a 3U CubeSat system architecture
  • Conducted hardware-in-the-loop testing using the RDP (Real-Time Dynamics Processor) and COSMOS interface
  • Configured and operated UUT/RDP/COSMOS test setups for closed-loop ADCS validation
  • Executed simulation-based test cases to evaluate pointing performance and system response
  • Extracted and processed telemetry data from COSMOS for analysis in MATLAB
  • Verified system performance against mission requirements defined in the Requirement Verification Matrix (RVM)
  • Collaborated across mechanical, electrical, and software teams to ensure proper subsystem interfacing
  • Supported command and telemetry pipeline development for ADCS–CDH interaction

Key features

  • Hardware-in-the-loop simulation: Testing with flight-representative ADCS hardware
  • Commercial ADCS integration: XACT-15 incorporated into a CubeSat architecture
  • Closed-loop attitude validation: High-fidelity orbital simulation in the loop with real hardware
  • Telemetry workflow: Extraction and post-processing for performance analysis
  • Requirements verification: Mission-driven pointing and stability targets traced to the RVM
  • Cross-disciplinary integration: ADCS, CDH, and supporting subsystem interfacing
